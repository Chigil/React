# NPM Base

## Short Description

You are to try common NPM tasks.

## Estimation (h)

16

## Topics

1. Package publishing
2. Autotests
3. Linting
4. Starting a web server
5. Compiling

## Requirements

* Publish your js lib as npm package.
* Cover it with autotests.
* Add linting.
* Override a few linting rules.
* Start a web server.
* Compile sass or less.
